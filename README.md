## Requirements

* PostgreSQL
* Ruby (>= 2.0)
* curl / curl dev libs

## Installation

Environment variables (`.env`):

* [YouTube API] key(https://cloud.google.com/console/project)
* [Steam API] key(http://steamcommunity.com/dev)
* [New Relic] key(http://newrelic.com/)
* Secret Token

`rake tf2movies:check`
