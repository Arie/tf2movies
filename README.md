## Requirements

* PostgreSQL
* Ruby 2.1
* Redis
* curl / curl dev libs

## Installation

* YouTube API key in `/config/initializers/youtube.rb`
* Steam API key in `config/initializers/omniauth_steam.rb`
* New Relic License key in `config/newrelic.yml`
